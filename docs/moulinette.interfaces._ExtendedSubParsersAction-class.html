Class _ExtendedSubParsersAction

source code

Subparsers with extended properties for argparse

It provides the following additional properties at initialization, e.g. using `parser.add_subparsers`:

  • required -- Either the subparser is required or not (default: False)

It also provides the following additional properties for parsers, e.g. using `subparsers.add_parser`:

  • deprecated -- Wether the command is deprecated
  • deprecated_alias -- A list of deprecated command alias names
