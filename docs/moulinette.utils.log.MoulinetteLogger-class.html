Class MoulinetteLogger

source code

Custom logger class

Extend base Logger class to provide the SUCCESS custom log level with a convenient logging method. It also consider an optionnal action_id which corresponds to the associated logged action. It is added to the LogRecord extra and can be used with the ActionFilter.
