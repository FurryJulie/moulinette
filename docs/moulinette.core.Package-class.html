get_cachedir(self, subdir='', make_dir=True)

source code 

Get the path to a cache directory

Return the path to the cache directory from an optional subdirectory and create it if needed.

Keyword arguments:

  • subdir -- A cache subdirectory
  • make_dir -- False to not make directory if it not exists

open_cachefile(self, filename, mode='r', **kwargs)

source code 

Open a cache file and return a stream

Attempt to open in 'mode' the cache file 'filename' from the default cache directory and in the subdirectory 'subdir' if given. Directories are created if needed and a stream is returned if the file can be written.

Keyword arguments:

  • filename -- The cache filename
  • mode -- The mode in which the file is opened
  • **kwargs -- Optional arguments for get_cachedir
