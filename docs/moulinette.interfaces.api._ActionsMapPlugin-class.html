Class _ActionsMapPlugin

Actions map Bottle Plugin

Process relevant action for the request using the actions map and manage authentication.

Keyword arguments:

  • actionsmap -- An ActionsMap instance
  • use_websocket -- If true, install a WebSocket on /messages in order to serve messages coming from the 'display' signal

Listen to the messages WebSocket stream

Retrieve the WebSocket stream and send to it each messages displayed by the core.MoulinetteSignals.display signal. They are JSON encoded as a dict { style: message }.
