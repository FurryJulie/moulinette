Class BaseActionsMapParser

source code

Actions map's base Parser

Each interfaces must implement an ActionsMapParser class derived from this class which must overrides virtual properties and methods. It is used to parse the main parts of the actions map (i.e. global arguments, categories and actions). It implements methods to set/get the global and actions configuration.

Keyword arguments:
